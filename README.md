# Game API as Monolit

### What do I have
These are the features that I've done for now

* Login API
  * POST /login
* Users API
  * GET /api/v1/users
  * GET /api/v1/users/{id}
  * POST /api/v1/users
  * PUT /api/v1/users/{id}
  * DELETE /api/v1/users/{id}
* User-Hero PI
  * GET /api/v1/users/{id}/heroes
* Heroes API
  * POST /api/v1/heroes
  * REMOVE /api/v1/heroes/{id}


### Techs
These are the technologies resources I'm currently using

* Java 17
* JWT
* Spring Boot 2.7.0
* Spring Security
